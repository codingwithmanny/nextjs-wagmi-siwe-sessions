# NextJS WAGMI SIWE Sessions

A small NextJS application/website that demonstrates SIWE with sessions.

If you like this check me out on Twitter at [@codingwithmanny](https://twitter.com/codingwithmanny)

## Requirements

- NVM or Node 16.17.0

## Local Setup

```bash
yarn install;
# npm install;
```

## Running Script

```bash
yarn dev;
# npm dev;
```
